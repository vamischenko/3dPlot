3dPlot
---------------------------------------------------------------------------------------------------
Построение графика и сохранение отчета:
1. Во вкладке "Data source" выбираем источник данных: рандом или CSV. Тут же можно и посмотреть считанные/сгенерированные данные.
ВНИМАНИЕ! Имя CSV вводится С РАСШИРЕНИЕМ + если файл не в папке с программой (проектом), то полный путь к файлу.
Если в CSV введен просто массив, то поля, связанные с CSV, не трогаем (они должны быть: 0, 0, пусто; кол-во заполнится автоматически после считывания).
Разделители между ячейками в CSV - точка с запятой.
2. Во вкладке "Analysis and plot" нажимаем "Analyse" и ждем. Входные данные будут обработаны.
Для построения графика жмем "Build plot".
Посмотреть срезы можно с помощью кнопок "View face" и "Build face". Для срезов также необходимо выбрать плоскость (точнее, ось):
    "Layer (Y-axis)" - по оси Y, матрица с анализом входного числа;
    "X-axis" - по оси X;
    "Z-axis" - по оси Z.
"Крутить" график можно во вкладке "View".
3. Во вкладке "Report" в поле "File name" вводим имя файла БЕЗ РАСШИРЕНИЯ (это не критично, просто оно будет добавлено в любом случае) и жмем одну из кнопок:
"Save to TXT" - сохранить в обычный текстовый файл или "Save to CSV" - сохранить в CSV.
4. PROFIT!
---------------------------------------------------------------------------------------------------
Далее по вкладкам:
-------- Data source --------------
"Generate data" - генерация массива случайных чисел:
    + "Count" - количество;
    + "Max value" - макс. значение одного числа;
    + "Generate" - генерация и установка сгенерированного набора как источник данных.
        ВНИМАНИЕ! ГЕНЕРИТЬ БОЛЕЕ 550 НЕ РЕКОМЕНДУЕТСЯ! ПРИ АНАЛИЗЕ МОЖЕТ ПРОИЗОЙТИ ПЕРЕПОЛНЕНИЕ!
"Read from CSV" - считывание из CSV:
    + "File name" - имя CSV-файла;
    + "Col" - начальный столбец;
    + "Row" - начальный ряд;
    + "Count" - кол-во счит. чисел;
    + "Parse orientation" - считывание по:
        + "Read by COL" - столбец;
        + "Read by ROW" - ряд;
    + "Read from CSV" - считывание из CSV и установка считанного набора как источник данных;
    ---
    + "View source data" - просмотр входных (считанных или сгенерированных) данных.
    ---
ВНИМАНИЕ! В CSV-файле в считываемом ряду (столбце) должны быть ТОЛЬКО числа и БЕЗ ПУСТЫХ клеток !
-------- Analysis and plot --------
"Analyse and build plot" - анализ входных данных и построение графика:
    + "Analyse" - анализ входных данных;
    Необходимо дождаться закрытия окна.
    + "Build plot" - построение графика;
    ВНИМАНИЕ! СТРОИТЬ ГРАФИК ДЛЯ БОЛЕЕ 30 ЭЛЕМЕНТОВ НЕ РЕКОМЕНДУЕТСЯ! МОЖЕТ ПРОИЗОЙТИ ПЕРЕПОЛНЕНИЕ!
    (30 элементов - можно строить, но потребуется довольно много ресурсов;
    50 элементов - произойдет переполнение)
    "Layer (Y-axis)" - срез по оси Y;
    + "X-axis" - срез по оси X;
    + "Z-axis" - срез по оси Z;
    + "Item number" - ввод номера элемента, для которого необходимо просмотреть срез;
    + "View face" - просмотреть срез в отдельном окне как матрицу;
    + "Build face" - - построить срез без остальных матриц.
-------- Report -------------------
"Report" - сохранение отчета (входные данные + матрица для каждого элемента):
    + "File name" - имя выходного файла;
    + "Save to TXT" - сохранить отчет в текстовый файл;
    + "Save to CSV" - сохранить отчет в CSV;
    + "Save to multiple TXT" - сохранить отчет в множество текстовых файлов (1 число - файл + файл с анализируемыми числами);
    + "Save faces by Y" - сохранить сечения, идущие по OY;
    + "Save faces by X" - сохранить сечения, идущие по OX;
    + "Save faces by Z" - сохранить сечения, идущие по OZ.
-------- View ---------------------
"View" - отдаление графика, его углы поворота:
    + "Distance by Z"
    + "By Z" - поворот по OZ;
    + "By X" - поворот по OX;
    + "By Y" - поворот по OY;
    + "Plot top" - вид на 0-вую матрицу спереди;
    + "Reset rotations" - сброс углов поворота.
